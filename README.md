# Sorting Algorithm Visualizer

A fully functional and interactive **Sorting Algorithm Visualizer** built using **HTML, CSS, and JavaScript**. This project helps users understand how different sorting algorithms work through animated visual representation of array elements.

## 🚀 Project Overview

Sorting Algorithm Visualizer is a web-based application that visually demonstrates the step-by-step working of popular sorting algorithms. The array is represented using vertical bars, where the height of each bar represents its value. During sorting, bars are highlighted to show comparison, swapping, and final sorted positions.

This project is useful for students, beginners, and anyone who wants to understand sorting algorithms in an easy and interactive way.

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## ✨ Features

* Generate a new random array
* Change array size using slider
* Control visualization speed
* Visual representation using bars
* Interactive sorting animation
* Clean and user-friendly interface
* Responsive design
* Stop or reset sorting process

## 📌 Sorting Algorithms Included

## 💡 How It Works

The visualizer creates a random array and displays each value as a vertical bar. When a sorting algorithm is selected, the algorithm runs step by step. The bars change color during comparison, swapping, and sorting, making it easier to understand the internal working of each algorithm.

## 🎯 Purpose of the Project

The main purpose of this project is to make sorting algorithms easier to understand through visualization. Instead of only reading theoretical concepts, users can watch how sorting happens in real time.
